3.Chef has his lunch only between 1 pm and 4 pm (both inclusive).Given that the current time is X pm, find out whether it is lunchtime for Chef.
code:
import java.util.Scanner;
public class Main{
    public static void main(String[]args) {
        Scanner sc=new Scanner(System.in);
        int T=sc.nextInt();
        while(T-->0) {
            int X=sc.nextInt();
            System.out.println((X>=1&&X<=4)?"YES":"NO");
        }
    }
}
