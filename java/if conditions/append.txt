10.Three digit number n is given. Append to it the digit 3 from the left and from the right.
code:
import java.util.Scanner;
public class Main {
    public static void main(String[]args) {
        Scanner sc=new Scanner(System.in);
        String n=sc.next();   
        System.out.println("3"+n+"3");
        sc.close();
    }
}
